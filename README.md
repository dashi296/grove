# Grove 🌿

**Notes That Grow With You**

Groveは、思考を整理するためのパーソナルメモアプリです。シンプルに使い始めて、必要に応じて拡張できます。

現在は初期開発段階です。このREADMEには、確定している設計方針と開発予定の機能を含みます。

---

## なぜGroveか

| Obsidianの不満 | Groveの解決策 |
|---|---|
| 起動・動作が重い | Tauri（Rust）で爆速起動 |
| UIが複雑すぎる | 必要な機能を内蔵、迷わないシンプルUI |
| 同期が有料・面倒 | 好きなストレージをプラグインで選択 |
| AI統合が弱い | AIをプラグインでネイティブ拡張 |
| モバイルUXが劣化版 | モバイルを独立設計する |

## 特徴

### ローカルファースト

ノートはMarkdownファイルとしてデバイスに保存する予定です。クラウドへの登録なしで使い始められ、アプリを使わなくなってもファイルが手元に残る体験を目指しています。

### シンプルなエディタ

デスクトップでは CodeMirror 6 をベースにした Markdown エディタを予定しています。モバイルは共通の command / event 契約を使いつつ、必要に応じて別実装を許容します。`[[WikiLink]]` によるノート間リンクとバックリンク生成も実装対象です。

### 高速な全文検索

SQLite FTS5による全文検索を計画しており、日本語を含むノート全体を高速に検索できるようにする方針です。

### 同期はプラグインで選ぶ

同期はデフォルトで無効にする予定です。必要になったときに、自分の環境に合ったプロバイダをプラグインとして追加できる形を目指しています。R2 や iCloud のような同期連携は本体に同梱せず、おすすめプラグインとして提示する想定です。iCloud については通常のクラウド API ではなくプラットフォームのフォルダ共有方式で扱うため、`sync-icloud` プラグインが利用する host 側の workspace file provider 境界を整備します。

## 同期プラグイン

同期プラグインは将来的に、設定画面のプラグインストアから追加できるようにする予定です。

### おすすめ

| プラグイン | 対応環境 | 特徴 |
|---|---|---|
| Cloudflare R2 | 全環境 | 安価・自己所有・E2E暗号化 |
| iCloud Drive | macOS / iOS のみ | Apple環境との自然な統合。クラウド API ではなく、host 側 file provider を使う `sync-icloud` プラグインとして扱う |

どのプラグインを使っても、ノートの中身はデバイス上で暗号化してからクラウドに送る設計です。サービス側は暗号文のみを扱います。

### コミュニティ（将来対応）

| プラグイン | 特徴 |
|---|---|
| Google Drive | Android環境で馴染みがある |
| Dropbox | ブロックレベル同期・API充実 |
| 汎用S3互換 | Wasabi・Backblaze B2・自前MinIO |
| Syncthing | P2P・クラウド不要・完全セルフホスト |
